# ☕ Java Code Centre

> *Write once. Use forever.*

A curated collection of **runnable Java programs** demonstrating Java concepts through clean, executable examples.

This repository is built as a long-term knowledge base for learning, revision, interview preparation, and project development. Every topic is organized into individual Java files that can be copied, executed, and modified without additional setup.
